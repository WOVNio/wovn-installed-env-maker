# MovableType

## 環境作成

1. `make` で環境の構築
2. `docker-compose up -d` でサービスの起動
3. http://127.0.0.1:8080/cgi-bin/mt/mt.cgi?__mode=rebuild_confirm&blog_id=1 にアクセスしてサイトの再構築
